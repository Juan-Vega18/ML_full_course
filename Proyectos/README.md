## Estructura de Cada Proyecto

Cada proyecto sigue esta estructura estándar:

```
PXX_nombre_proyecto/
├── README.md               # Descripción del problema, metodología y resultados
├── nombre_proyecto.ipynb   # Notebook principal con todo el análisis
└── data/                   # (ignorada por git) — dataset local
```

---

## Cómo Contribuir un Nuevo Proyecto

1. Crear una carpeta `PXX_nombre_proyecto/` dentro de `Proyectos/`.
2. Añadir un `README.md` describiendo el problema, los datos y los resultados obtenidos.
3. Desarrollar el análisis en un Jupyter Notebook bien documentado.
4. Actualizar este `README.md` con el nuevo proyecto en la tabla.
