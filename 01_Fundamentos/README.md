## Conceptos Clave

- **Machine Learning**: sistemas que aprenden de los datos sin ser programados explícitamente.
- **Tipos de ML**:
  - *Supervisado*: el modelo aprende de datos etiquetados (entrada → salida conocida).
  - *No supervisado*: el modelo encuentra patrones en datos sin etiquetar.
  - *Por refuerzo*: el agente aprende a través de recompensas y penalizaciones.
- **Flujo típico de un proyecto ML**:
  1. Definir el problema
  2. Recolectar y explorar datos (EDA)
  3. Preprocesar datos
  4. Seleccionar y entrenar el modelo
  5. Evaluar y ajustar
  6. Desplegar

## Herramientas

- **NumPy**: operaciones vectoriales y matriciales de alta eficiencia.
- **Pandas**: DataFrames para manipulación y análisis de datos tabulares.
- **Matplotlib / Seaborn**: bibliotecas de visualización estática y estadística.
