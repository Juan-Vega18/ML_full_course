## Conceptos Clave

- **Neurona artificial**: recibe entradas, aplica una función de activación y produce una salida.
- **Función de activación**: ReLU, Sigmoid, Tanh, Softmax.
- **Backpropagation**: algoritmo para calcular gradientes y actualizar los pesos.
- **Gradient Descent / Adam**: optimizadores para minimizar la función de pérdida.
- **Épocas y batch size**: hiperparámetros del entrenamiento.
- **Dropout / Regularización L2**: técnicas para evitar overfitting.
- **CNN**: capas convolucionales para detectar patrones espaciales en imágenes.
